# MenuBar YouTube
## add Icon in macOS MenuBar and play YouTube video on popover view

write the Movies 
~~~
settings.json
{
    // Here Set the display videos Max:4 Min:1
    "max_display_video":"4",
    // Here Set the display video's width
    "video_width":"460",
    // Here Set the display video's height
    "video_height":"180"
}
~~~
